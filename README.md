# TCC_POLI_USP_Data_Science_-_Analytics_2026
## Modelo econométrico para analisar o impacto de variáveis econômicas no preço de eletrodomésticos no Brasil

Este trabalho apresenta um modelo econométrico para
analisar o impacto de variáveis econômicas no preço médio
de eletrodomésticos no Brasil, um setor que reflete de
maneira sensível as oscilações macroeconômicas e
representa uma parte relevante do consumo das famílias.
Diante da lacuna de estudos quantitativos precisos, o
objetivo geral foi avaliar quantitativamente a influência
de variáveis econômicas selecionadas, como Selic, Índice
de Preços ao Produtor (IPP), Massa Salarial e Índice de
Confiança do Consumidor (ICC), sobre o preço médio de
Lavadoras Automáticas Topload, Fogões e Refrigeradores
Topfreezer. A metodologia consistiu em uma análise
exploratória das variáveis macroeconômicas em séries
temporais e as respectivas correlações de Pearson com a
variável estudada. Em seguida foi desenvolvido modelos de
regressão linear múltipla, utilizando dados de 51 meses
(Janeiro de 2021 a Março de 2025). Os modelos definitivos
ajustados apresentaram elevado poder explicativo, com R²
ajustado superior a 0.94 e baixo Erro Absoluto Médio (MAE)
para todas as categorias. Os resultados indicaram que o
IPP foi a única variável significativa para o preço de
todas as três categorias, comprovando a relação
custo-insumo e preço de venda. A Selic e o IPP
influenciaram significativamente o preço dos bens de maior
valor (Lavadora Topload e Refrigerador Topfreezer). Por
fim, a conclusão é que a dinâmica de preço é distinta por
categoria.
